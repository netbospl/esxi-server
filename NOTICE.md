# AI-Assisted / Vibe-Coded Notice

This repository is AI-assisted / vibe-coded. It is an experimental ESXi Server Skill intended to help humans and AI agents reason about VMware ESXi operations, not a substitute for operator judgment.

Commands, API examples, and operational guidance must be reviewed by a human before use on real ESXi hosts. The skill may contain incomplete assumptions, environment-specific details, or rough edges, especially where standalone ESXi behavior differs from vCenter-managed environments.

Do not run destructive actions without explicit confirmation and a clear rollback plan. This includes deleting VMs, deleting snapshots, removing datastore files, changing networking, reverting snapshots, or powering off production workloads.

No warranty is provided. Operators remain responsible for validating commands, reviewing safety implications, and adapting procedures to their own ESXi environment.
